## Core Features

### Git-Style Conversation Branching

- **Manage Chat Like Code:** Create a "New Branch" at any node to explore different paths while keeping the main thread clean.

- **Spatial Navigation:** Click any node in the graph to instantly scroll the main window to the corresponding conversation block.
    

### 🎲 Semantic Neural Graph [Under Developing]

 - Use 2D Network to show the connections between each prompt. Provides an overall display of the entire conversation.

### 🎨 High-End Interaction Design

- Move nodes between branches with a "Predictive Ghost Line" preview to reshape your thinking topology in real-time.

## 🛠️ Tech Stack

- **Framework:** Vite + React + Tailwind CSS
- **Graphics Engine:** D3.js (Force-directed layout with collision detection)
- **Natural Language Processing:** [Transformers.js](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fxenova%2Ftransformers.js) (Model: all-MiniLM-L6-v2)
- **Storage:** Chrome Storage API + IndexedDB (for large-scale vector storage)
- **Injection Logic:** High-performance MutationObserver + Cross-world communication (Isolated -> Main World)

## 📦 Installation

1. **Clone the Repository:**
```    
git clone https://github.com/devdede12345/Better-Google-AI-Studio-GUI.git
```
2. **Load the Extension:**
    - Open Chrome and navigate to chrome://extensions/.
    - Enable **"Developer mode"** in the top right corner.
    - Click **"Load unpacked"**.
    - Select the project folder you just cloned.
3. **Launch:**
    - Visit [Google AI Studio](https://www.google.com/url?sa=E&q=https%3A%2F%2Faistudio.google.com%2F).
    - Click the **🌿 (Git Icon)** on the left sidebar to open the AetherMind pane
